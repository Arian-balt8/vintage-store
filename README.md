# Vintage Store - Página Web

## Descripción
Este proyecto consiste en la creación de una página web sobre una tienda de ropa vintage. 
El objetivo del proyecto es aplicar trabajo colaborativo utilizando Git y GitHub, 
incluyendo commits, ramas, pull requests y merge.

## Trabajo colaborativo
Se utilizaron ramas para el desarrollo del proyecto:
- main
- index
- productos
- contacto
- estilos
- javascript

Cada integrante trabajó en su rama y luego se realizaron Pull Requests para unir los cambios al proyecto principal.
